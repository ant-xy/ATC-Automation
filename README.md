# Cloning

```
git clone https://github.com/ant-xy/ATC-Automation.git &&
cd ATC-Automation &&
python3 -m venv .env &&
source .env/bin/activate
```

# Installation
## Linux
```
python -m pip install -r requirements.txt
```

## Windows
```
py -m pip install -r requirements.txt
```

# Running

```
python3 main.py
```
